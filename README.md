## Installation Web

Important: Use node version 10.15.0

### Run web in dev mode with local ganache

```bash
$ npm run web:local
```

## Testing

We are using [snapshot testing](https://jestjs.io/docs/en/snapshot-testing)

```bash
# Web
$ npm run test:web

# Update Snapshots
$ npm run test:web -- -u

# Web watch mode
$ npm run test:web-watch

# Coverage - web
$ npm run coverage
```
